# BrainfuckIDE
## これはなに？
Breinf\*ckの統合開発環境を目指して作りました。  
主にデバッグ環境

## 何ができるの？
Brainf\*ckのコードをブレイクポイントで停止しながらデバッグできます。  
メモリの書き換えやを行うことができ　スニペットも基本的な演算に関しては実装しました。
